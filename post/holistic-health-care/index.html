全人關懷實踐工作坊

作者 / 王怡涵營養師     2023/01/07

在醫療場域中，看到候診的叫號人數不斷累積或是開啟病房會診清單跳出長長的一頁，在緊湊的節奏與壓力中，閱讀與查詢個案的電子病歷後，詢問其飲食習慣，給予飲食與生活型態調整的建議。但這樣做真的有幫助到個案嗎? 個案會按照衛教單張上的建議改變嗎?

曾經在神經外科加護病房照護頭部外傷的重症病人(中壯年男性)時，因其狀況不穩定，使用鼻胃管灌食，但因腦壓高與消化差等問題，其營養品灌食量少(第二週時每天約攝取500–750大卡/天)。病人的媽媽想使用親友推薦的其他產品(非衛福部核可)，來幫忙增加病人的營養與免疫力。當時我與媽媽，花了一些時間討論，和她解釋親友推薦的產品之熱量、蛋白質、營養素很少，而病人現在使用的灌食為高蛋白配方，也有維生素與礦物質等，對病人來說是營養密度高、很營養的配方。

討論完的幾天後，加護病房護理師打電話來說，病人媽媽還是堅持要使用親友推薦的產品，而且從原本要求的1天1瓶增加到2瓶。

現在回想當下的我，沒有照顧到母親的心靈，她希望竭盡所有資源、找各種方法讓她的孩子可以變好，而我疏漏照顧母親的心靈，讓她處於幫不上忙的焦慮無力、擔心無助中，同時討論所花費的時間似乎沒有讓媽媽的想法產生改變。

透過在醫院的全人教師培訓課程與跨院工作坊
而認識全人照護與靈性關懷

全人照護是什麼？

  • 導入靈性的全人照護、全家、全程、全團隊、全社區的照護
  • 四個指標
    1. 生理：疼痛指數、身體症狀、生活功能、存活率預估
    2. 心理：情緒狀況、憂鬱狀況、失眠、有無自殺念頭
    3. 社交：人際關係、朋友、重要他人、信仰
    4. 靈性：靈性的信念、實踐、經驗

靈性是什麼？

靈性的內涵(五面向) 生命中，什麼對你而言是有意義或目的?

  1. 發現或尋求生命意義
    — 生命中，什麼對你而言是有意義或目的?
    — 如果你有機會讓生命重來，你想要過怎樣不同的生活?
    — 當這些狀況(如生命末期或痛苦)發生時，對你有何意義?
  2. 經歷希望/盼望
  3. 愛與被愛
    — 失去的要如何補回?
    — 這一聲中你最愛最放不下的人是誰呢?
    — 你認為最捨不得你或疼愛的人是誰呢?
  4. 寬恕或被寬恕
    — 誰曾傷害了你? 你曾傷害了誰嗎?
  5. 與至高者的關係(信仰)

如何進行靈性會談?

  1. 初步問好、與病人建立談話的關係
  2. 發現承認問題
  3. 深入探討問題
  4. 找到解決方法

靈性關懷的三階段方法

第一階段：自我承認 (靈性鑑別診斷-靈性評估)

第二階段：深入探討原因 (尋找致病因-靈性會談)

第三階段：找到解決方式 (提供處方-靈性介入)

反思與覺察－若回到那天

如果當時的我認識全人照護與靈性關懷，我想同理這位擔心焦慮的母親，和她說：我感受到你對兒子的愛以及妳非常擔心他營養不夠會影響他的病況，妳想給他很好的營養，這非常的重要，那我們一起來看看親友推薦的產品和醫院現在正在使用的營養品配方，裡面的營養成分有哪些，現在他因為消化差，灌進去的量有限，我們一起討論哪些對他來說是目前最需要的營養。

提醒自己不僅照護病人，也要覺察與感受病人家人們的身心靈，在他們感到無助難過時，能帶給他們些許的溫暖與支持
