動機式晤談工作坊

作者 / 王怡涵營養師     2022/11/15

在門診飲食諮詢時，當個案聽到飲食調整的方法，其當下出現抗拒不願意改變或是說知道怎麼做但做不到時，讓我感到無所適從，不知道如何幫助個案有動機做改變。

某次機會下透過朋友的推薦與分享，在營養師PGY時，讀了陳偉任醫師的書「強化動機 承諾改變：動機式晤談實務工作手冊」，學習有許多方法可以增加個案的動機。在門診中練習應用其中的一些技巧，看見有的個案的眼中發出的光芒與開心的笑容，讓我發現除了以前學習書本中的疾病飲食建議原則外，更重要的是讓個案增加動機，並提供適合他飲食與生活型態的調整方法。除了書之外，透過2022/10/29–30的動機式晤談工作坊課程的實際演練，以及陳偉任醫師的示範講解，更能了解動機式晤談的精神與想傳遞的理念。期望在每次與個案討論的過程中，能為其生活帶來一些幫助，一起往更好的地方前進。

我們每個人都希望好好被對待與溝通，在認識動機式晤談以前，常常在諮詢的過程中不小心落入對話陷阱，封閉式的提問(如體重有沒有增加?有沒有規律吃藥?)、專家(研究顯示…,我的建議是…)、過早聚焦(是因為喝飲料所以血糖高)、標籤(你血糖控制不太好,是不是飲食不規律)、責難、閒聊等等。

為了解決個案的問題，常常急著給予介入的建議與方法，彷彿趕緊告訴他方法就可以減少我心中的焦慮以及門診看診的時間壓力。但往往忘記坐在眼前的人是需要被理解與尊重，要解決的可能不只是問題或抽血報告上的紅字，而是讓他看見自己有力量可以改變，可以變得更好。透過對話的過程，引導個案思考，並探索他的期待與渴望，幫助他點燃心中的燈，陪伴他走出深邃的黑暗隧道，到達他想去的地方。

動機式晤談會帶來什麼幫助?

  • 讓個案燃起改變的動機

  • 與個案的互動變好，像朋友般的老師陪伴他面對困擾著他的問題

  • 成為有溫度的助人工作者

動機式晤談的精神

  • 合作：像朋友般的老師，邀請個案一起看到問題、矛盾

  • 誘發：個案抗拒時邀請個案看到不一致

  • 自主：讓個案說出他想改變、自己決定改變

動機式晤談的技巧 — OARS

Open question：開放式問句
Affirmation：給予肯定
Reflection：反映式傾聽
Summary：摘要

  • 簡單回映：與個案說的內容大致相同，讓個案感覺有被聆聽

  • 複雜回映：將個案說的內容換句話說，並幫個案說出背後可能有的感覺如無力、焦慮、挫折等

  • 放大回映：將個案說的內容放大解釋，如我很忙沒有空運動→你似乎擠不出任何一點時間運動

  • 兩面回映：點出個案的矛盾，如一方面你知道運動對你很重要，另一方面你卻擠不出時間做運動

  • 隱喻：和個案說故事-與個案情境類似，讓個案思考與燃起改變的勇氣

助人改變的歷程 (諮詢的步驟)

  1. 融入：建立與個案的互信與尊重關係，以個案為中心的傾聽
  2. 聚焦：邀請個案設定討論議題
  3. 誘發

運用改變的對話的技巧，評估個案目前的動機為何，給予適當的回應並觸發其改變；並在對話的過程中持續觀察個案語言或非語言的表達，以評估其動機如何(觀察個案處於抗拒/矛盾/有動機缺方法的哪個狀態，依其狀態給予合適的回應)

抗拒-處理的策略

  • 簡單回映：與個案說的內容大致相同，讓個案感覺有被聆聽

  • 放大回應：將個案說的內容放大解釋，如我很忙沒有空運動→你似乎擠不出任何一點時間運動

  • 兩面回應：點出個案的矛盾，如一方面你知道運動對你很重要，另一方面你卻擠不出時間做運動

  • 移開焦點：詢問個案對於問題家人的看法？如家人如何對你的血糖/喝酒有什麼想法？

  • 稍加更動後同意

  • 重新框架：如個案說我來聽過好多次衛教，血糖都很高知道怎麼做但我都還是做不到。這時候可以詢問個案，是什麼力量讓你願意花時間坐在這裡努力找讓血糖變好的方法？同時可以觀察以前他的血糖變化，有比較好的時候可以詢問你是怎麼做到的？並以鼓勵與肯定的正向態度協助他改變

  • 強調個人選擇

矛盾-處理的策略

  • 直接詢問

  • 擴大說明：舉例子、示範典型的一天，刻意掛鉤與人的關係，該路重點，點出個案矛盾處

  • 回顧過去：以前沒有問題或問題相對少的時候，那段生活是如何的？

  • 想像未來：假如這個問題從你身上離開了，你的生活會有什麼不一樣？

  • 量尺問句

  • 正反雙方考量：拿紙筆幫個案寫下不改變與改變的好處與代價

  • 使用極致：如果這件事沒有改，最糟糕的情況(或1年後)會變怎麼樣？

  • 評估回饋：邀請個案表達對這份報告的看法，如假如你知道報告的內容，你會覺得長怎麼樣？(評估個案的動機與期待)

  • 探索價值：改變的動力一直都在每個人的心中，只是暫時沒看到，可以問個案3–5個為什麼，為什麼那麼重要(讓個案了解他有選擇的自由、擁有足夠的改變力量、專注於各任何微弱的動力跡象並幫助強化)

有動機-進入發展計畫

  1. 發展計畫：協助個案改變與發展改變計劃

提供訊息三步驟：尋求個案同意 → 提供多項選擇 → 詢問個案的看法

單次助人工作模式的步驟

  1. 最近有什麼問題困擾你？

  2. 你有多想達成這個目標？這裡有一把量尺從0–10分。0分是你一點都不想，10分是你分常想要。目前的狀況你會選擇幾分？

  3. 為什麼你沒有選擇比＿分更低一點的分數？

  4. 如果OO問題獲得解決，不再困擾你，那你的生活將會有什麼不一樣？

  5. 這些不一樣為什麼對你這麼重要？

  6. 我把剛才聽到的做個整理，我聽到你…(問題)，你也很想…(目標)，因為你很在意(改變的動力)，不知道這樣的瞭解對不對？

  7. 帶著剛才這樣的了解，第一步你要做些什麼才能從＿分進步到＿分？若個案想不到改變的做法，可應用提供訊息三步驟技巧，尋求個案同意後，給予多項選擇再詢問其看法

  8. 讓我說說看我對你的了解是否正確？你提到…。我有沒有遺漏了什麼？還是有什麼地方需要做修正？
