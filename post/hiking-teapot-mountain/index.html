無耳茶壺山、報時山步道

作者 / 王怡涵營養師     2021/06/04

交通

去程：搭965公車，捷運站北門站 → 瓜山國小
回程：搭965公車，九份老街 → 捷運站北門站

步道圖

路線

瓜山國小 → 勸濟堂→ 報時山步道 → 無耳茶壺山 → 勸濟堂→ 祈堂老街 → 黃金博物館

瓜山國小 → 勸濟堂

下車到瓜山國小後往前走一點會看到路標

勸濟堂

報時山步道

報時山步道→ 無耳茶壺山

網美打卡景點 ? (坐在殘壁上看起來好危險啊)

無耳茶壺山登山口

無耳茶壺山

途中有碎石路段，一定要穿好走的登山鞋或運動鞋保護腳踝

如果不方便走久的話，可以開車沿著道路開到登山口，在涼亭看風景也很棒 ~

停在登山口附近的車子變得很小

從寶獅亭眺望的全景

平日也有不少人在爬山

怎麼拍怎麼美的風景

無耳茶壺山

這段路稍微要拉繩子攀爬，右邊的洞一次只能一個人通過，很狹窄

← 有勇敢的人攀到這顆岩石，爬到最高點拍照 (但我不敢😂)

寶獅亭

從寶獅亭可以看到瓜山國小的操場和金瓜石緩慢民宿

祈堂老街

阿婆雜貨店（找不到阿婆在哪裡

靜謐的老街街道

散散步咖啡（只開五六日13:30~18:30，今天竟然是週四😭）

祈堂老街→ 黃金博物館

原來祈堂老街和黃金博物館只距離180公尺

基隆山，又稱大肚美人山

沿著階梯繼續往上走，就會到黃金博物館內瑞芳分局附近的步道

瑞芳分局

紀錄

瓜山國小 → 報時山步道 → 登上無耳茶壺山

共約1.5小時、4公里、爬升500公尺

無耳茶壺山下山→ 祈堂老街 → 黃金博物館

共約45分鐘、2公里

報時山步道和無耳茶壺山的天空和海超級美，爆汗爬山很值得，算是易爬的山（除了最後一段攀岩之外😂）

一路上偶爾有樹蔭，途經的涼亭有3個，大致上非常的熱，一定要帶足夠的水、食物、毛巾以及做好防曬，如果風不大，撐傘爬山會比較舒服，避免天熱中暑的危險。
